# Орындалған (зерттелген) процестердің, процедуралардың, әдістердің т. б. Сипаттамасы

## Жоба: SmartCourse — AI негізіндегі курстарды басқару жүйесі

**Технологиялық стек:**
- **Backend:** Go (Gin фреймворкі), GORM ORM, PostgreSQL, Redis, MinIO
- **Frontend:** Next.js 16, React 19, TailwindCSS, shadcn/ui, Recharts, FullCalendar
- **AI:** OpenRouter API (GPT), Google Gemini AI (Chat, TTS, Audio Understanding)
- **DevOps:** Docker Compose, Prometheus, Grafana
- **Аутентификация:** JWT (JSON Web Token), bcrypt
